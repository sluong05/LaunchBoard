# Launchboard

A React quick-links dashboard for collecting the links you need every day.

## Features

- Add, edit, delete, and favorite links
- Add optional notes so each link explains where it goes or what to do there
- Organize links by preset or custom section and color
- Search across titles, URLs, notes, sections, and tags
- Open the currently shown links at once
- Import and export your link collection as JSON
- Persists locally in the browser

## Run Locally

```bash
npm install
npm run dev
```
