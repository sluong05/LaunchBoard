# Launchboard

A React quick-links dashboard for collecting the links you need every day.

## Features

- Add, edit, delete, and favorite links
- Organize links by section and color
- Search across titles, URLs, sections, and tags
- Open the currently shown links at once
- Import and export your link collection as JSON
- Persists locally in the browser

## Run Locally

```bash
npm install
npm run dev
```
